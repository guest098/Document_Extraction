# Project Setup

## Installation
Install dependencies:

```bash
npm install

Environment Setup

Create a .env file in the project root and configure:

GOOGLE_GEMINI_API_KEY=your_gemini_api_key_here
MONGODB_URI=your_mongodb_connection_string_here
Run Development Server

Start the project:

npm run dev
