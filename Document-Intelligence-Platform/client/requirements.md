## Packages
react-pdf | For rendering PDF documents in the browser
recharts | For risk analysis charts and data visualization
react-dropzone | For drag and drop file upload
framer-motion | For smooth page transitions and UI animations
date-fns | For date formatting
clsx | For conditional class names
tailwind-merge | For merging tailwind classes safely

## Notes
Authentication uses the existing use-auth hook
PDF viewer requires careful worker configuration (handled in code)
File uploads use FormData with multipart/form-data
Backend API paths defined in @shared/routes
